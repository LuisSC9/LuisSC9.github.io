LISTAS EN HTML




						Las listas HTML permiten a los desarrolladores web 
						agrupar un conjunto de elementos relacionados en listas.
					









						Una lista desordenada comienza con la etiqueta,
						cada elemento de lista comienza con la etiqueta.
						Los elementos de la lista se marcarán con viñetas 
						(pequeños círculos negros) de forma predeterminada.
